## 四、剩余页面 Pencil 设计策略

### 策略说明
- 每个页面在 **同一个 .pen 文件**中作为独立 Frame
- 每个页面复刻侧边栏结构（保持一致导航）
- **先做 Dark 模式 Dashboard**，确认暗色效果
- **再做 Stats、Todo、Pomodoro、Notes** 四个功能页

### 页面设计要点

#### 🌓 仪表盘 Dark 模式
- 复制现有仪表盘 Frame，切换 `.dark` 主题变量
- 验证暗色下所有卡片、图表、文字对比度

#### 📈 统计详情页 `/stats`
- **顶部**：时间粒度切换 Tab — 日 / 周 / 月 / 年（滑动指示器动效）
- **摘要卡片行**：总时长 · 完成番茄数 · 日均 · 完成率
- **趋势图**：根据所选粒度切换柱状图数据（纯 SVG/CSS）
- **分类分布**：环形图 + 标签百分比列表
- **空状态**：无数据时的引导卡

#### 📋 待办页 `/todo`
- **顶部**：输入框 + 添加按钮
- **筛选栏**：全部 / 今天 / 计划中 / 已完成
- **任务列表**：复选框 + 标题 + 描述 + 标签（优先级色） + 番茄数
- **每一行**支持展开描述、关联笔记
- **动效**：勾选划线、删除滑出

#### 🍅 番茄钟页 `/pomodoro`
- **中央大圆环**：SVG 环形进度倒计时 + 大号数字
- **模式切换**：工作 25 / 短休 5 / 长休 15
- **控制区**：开始/暂停按钮（大圆形）+ 跳过 + 重置
- **底部**：关联任务 + 今日完成番茄数

#### 📝 笔记页 `/notes`
- **左栏**（约 280px）：笔记列表 + 搜索
- **右栏**：标题 + Markdown 编辑器 + 标签
- **列表项**：标题 + 时间 + 标签缩略
- **新建按钮**：列表底部居中

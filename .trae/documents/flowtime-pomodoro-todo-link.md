# FlowTime - 番茄钟 ↔ 待办 联动设计

## 联动模型（已确认）

### 核心规则：**可选关联任务**

番茄可以不关联任务（自由专注模式），也可以关联一个 Todo 任务。

```
任务（Todo）          番茄钟（Pomodoro）
┌─────────────┐      ┌──────────────┐
│ 标题        │◄──?──│ 关联任务      │  ← 可选关联
│ 描述        │      │ 计时器        │
│ 预计番茄数  │───→  │ 25:00        │
│ 已完成番茄  │      └──────────────┘
└─────────────┘
```

---

## 双向入口设计

| 入口 | 场景 | 行为 |
|------|------|------|
| **Todo 页** 点 `▶🍅` | "我决定做这个任务了" | 跳转到番茄钟页，自动关联该任务 |
| **番茄钟页** 底部任务卡 | 刚打开番茄钟 / 切换任务 | 显示"可选关联任务"，点击展开 Todo 列表选择 |
| **自由专注** | 只想专注，不管任务 | 任务卡显示"无关联任务 — 自由专注" |

---

## 数据流

```
番茄启动 → 可选项：关联任务 / 自由专注
番茄完成 → 若有关联任务 → task.completedPomos += 1
        → 若 completedPomos >= estimatedPomos → task.status = "done"
番茄跳过 → 创建未完成记录，不计入任务统计
番茄放弃 → 无任何数据变化
```
